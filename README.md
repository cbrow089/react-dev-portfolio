  ## Description
  * React Dev Portfolio is a REACT application using JavaScript that sets up a professional portfolio to display a users information on a web browser using  react-router-dom.

  ## Installation
  * This is a project using React packages inside node.js  it is built using vite.   The user would use the commaned 'npm create vite' inside the command line and will be prompted to set up a vite folder for the application.
    
  ## Usage
  This application is used by typing 'npm run dev' in the command line interface and it will automatically open the localhost port 3000 to render the webpage inside the browser.  This application is a stactic portfolio that does not accept user inputs.  It is a way for a user to display and post their portfolio or professional work and skillset to an online rendering for potential employers to be able to view easily.  This project is deployed via Netlify.
